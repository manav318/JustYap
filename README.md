# JustYap

JustYap is a project for quick and easy messaging. This repository is under active development.

## Features

- Real-time chat functionality
- User authentication
- Responsive UI

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/manav318/JustYap.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm start
   ```
